035/**
036     This Enum defines a set of constants, each of which specifies a
037     Unicode "block" of characters to be matched (see below).  Each
038     of these Enum constants can be used to instantiate a {@link BuiltInCharClass}
039     that matches any of the characters in the corresponding Unicode block.
040     For example,
041<p>
042<pre>
043      Pattern GreekCoptic =
044                new BuiltInCharClass(UniBlockBuiltIn.Greek_and_Coptic);
045</pre>
046     matches (on each repetition) one Greek or Coptic character.
047
048@see <a href="http://en.wikipedia.org/wiki/Unicode_block"> Unicode block</a>
049
050*/
