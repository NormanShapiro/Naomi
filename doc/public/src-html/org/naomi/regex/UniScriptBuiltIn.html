035/**
036     This Enum defines a set of constants, each of which specifies a
037     Unicode "script" of characters to be matched (see below).  Each
038     of these Enum constants can be used to instantiate a {@link BuiltInCharClass}
039     that matches any of the characters in the corresponding Unicode script.
040     For example,
041<p>
042<pre>
043      Pattern Arabic = new BuiltInCharClass(ScriptBuiltIn.ARABIC);
044</pre>
045     matches (on each repetition) one Unicode character in the ARABIC script.

047@see <a href="http://en.wikipedia.org/wiki/Script_(Unicode)"> Unicode script</a>
